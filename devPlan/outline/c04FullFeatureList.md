# 四、完整版功能清单

| 模块 | 功能 | 状态 | 备注 |
|------|------|------|------|
| 游戏流程 | 白天/夜晚交替、阶段切换 | ✅ 已完成 | 60s 白天 / 30s 夜晚自动循环，支持手动跳过 |
| 夜晚经济 | 商店购卡、花钱修复建筑 | 🟡 部分 | 购卡与拖拽直放已完成；建筑修复待 M3 |
| 夜晚休眠 | 无夜战词条的单位/建筑停止工作 | ✅ 已完成 | M1 门控骨架完成，M4 接入夜战例外 |
| 商店系统 | 购买建筑卡牌 | ✅ 已完成 | 刷新、购买、拖拽直放 |
| 手牌系统 | 卡牌管理、打出 | ✅ 已完成 | 点击与拖拽放置 |
| 建筑系统 | 玩家放置、多类型、产出 | 🟡 部分 | 玩家可放置兵营；多类型待 M3 |
| 战斗系统 | 部队 AI、攻击、死亡 | 🟡 部分 | 极简互殴已实现 |
| 邻接系统 | 建筑邻接加成 | ❌ 待开发 | M3 |
| 融合系统 | 建筑融合升级 | ❌ 待开发 | M5 |
| UI 界面 | 商店、手牌、金币、融合 | 🟡 部分 | 商店、手牌、金币、放置预览已有；菜单与信息面板待补 |
| AI 对手 | 购卡/放置/决策 | ❌ 待开发 | 敌方仅为对称自动兵营 |
| 数据层 | CardData / BuildingData 资源 | 🟡 部分 | `GameConfig`、`CardData` 运行时结构已有；`.tres` 资源待 M3 |
| 存档 | 保存/读取 | ❌ 待开发 | 不在 MVP 范围 |
